PolygonFiller - program for filling polygons with texture, height map and lighting
Author: Jakub Rymuza

Manual in polish:

Instrukcja obsługi:

dokładność triangulacji - suwak

wybór tekstury - radiobutton - po kliknięciu wyświetla się menu do wybrania koloru lub pliku
wybór NormalMap - przycisk wyświetla menu wyboru pliku

włączenie/wyłączenie animacja światła - checkbox
wysokość światła - suwak
kolor światła - przycisk wyświetla meny do wybrania koloru

metoda kolorowaniu - dokładne/interpolacja - radiobutton

współczynniki malowania (k, ks, kd, m) - suwaki

schowanie/pokazanie siatki trójkątów - przycisk
